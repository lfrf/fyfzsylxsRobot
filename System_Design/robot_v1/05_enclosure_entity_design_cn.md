# 桌面机器人外壳与实体模块总设计方案（V1）

## 1. 目标与约束

### 1.1 目标
1. 在不改动现有核心软件链路的前提下，完成机器人实体结构的可制造方案。
2. 外壳设计需要兼容已确定模块：树莓派、DS3218 双舵机、PCA9685、双 OLED、CSI 摄像头、reSpeaker Lite、扬声器、电源模块。
3. 支持快速装配、维护拆装与后续迭代。

### 1.2 约束
1. 形态：固定桌面机器人，无底盘移动。
2. 外形上限建议：`170(W) x 150(D) x 230(H) mm`。
3. 头部总质量建议：`<= 300g`。
4. 舵机供电与主控供电分离，结构需预留双电源与安全开关安装位。

## 2. 整机机械架构

## 2.1 分总成
1. A 总成：头部壳体总成（面罩、后盖、内部模块架）。
2. B 总成：颈部云台总成（双舵机 + 连接支架 + 限位件）。
3. C 总成：底座壳体总成（上盖、中框、底盖）。
4. D 总成：电气与线束总成（电源、驱动板、接插件）。

### 2.2 推荐装配关系
1. C 底座总成固定在桌面，内部完成电源与主控装配。
2. B 云台总成固定在底座顶部中心，作为头部转动机构。
3. A 头部总成通过俯仰轴连接到 B 总成上部。
4. D 线束从底座经颈部中空通道进入头部。

## 3. 外形与关键尺寸

### 3.1 外形尺寸（建议）
1. 整机：`170 x 150 x 230 mm`。
2. 底座外壳：`170 x 150 x 45 mm`。
3. 颈部机构区：`90 x 80 x 70 mm`（内部净空）。
4. 头部外壳：`145 x 120 x 105 mm`。

### 3.2 壳体厚度与结构
1. 外壳壁厚：2.4mm。
2. 加强筋厚：2.0mm。
3. 螺柱根部圆角：R2 及以上。
4. 铜螺母柱外径建议：`>= 7mm`。

### 3.3 坐标基准（用于统一孔位）
1. 全机基准原点 O：底座顶面中心点。
2. X 轴：左右方向，Y 轴：前后方向，Z 轴：竖直方向。
3. 所有安装孔建议使用相对 O 点的坐标标注，方便 CAD 与装配对齐。

## 5. 头部壳体设计

### 5.1 头部壳体组成
1. 前面罩（含双眼窗口与相机孔）。
2. 后盖（可拆维护盖，优先快拆）。
3. 内部设备托架（固定 OLED、相机、麦克风）。

### 5.2 前面板开孔建议
1. 左右眼窗口：按显示有效区 + 每边 1.5mm 遮边。
2. 摄像头孔：镜头外径 + 1.0mm 余量。
3. 麦克风开孔：条形阵列孔，单孔 1.5mm，孔距 2.5mm。
4. 前面板内侧建议加遮光罩，降低屏幕串光和镜头眩光。

### 5.3 头部内部布局（从前到后）
1. 最前层：双 OLED 与相机。
2. 中间层：麦克风板与走线桥架。
3. 后层：线束余量仓与检修空间。

### 5.4 维护设计
1. 后盖采用 4 颗 M3 螺丝固定。
2. 关键插头（OLED、摄像头、麦克风）设置防呆标签。
3. 后盖开启后可直接触达全部头部模块。

## 6. 颈部云台与转动结构

### 6.1 云台结构
1. 水平轴舵机安装在底座中心平台。
2. 俯仰轴舵机安装在上层 U 型支架。
3. 头部通过俯仰轴连接板固定。

### 6.2 机械限位
1. 水平机械限位建议：中心左右各约 75°。
2. 俯仰机械限位建议：下俯约 30°，上仰约 40°。
3. 软件限位小于机械限位 5° 以上，避免撞限位。

### 6.3 线束通道
1. 颈部中间预留 `>= 12mm` 线束孔。
2. 线束加蛇形余量，避免舵机全角度时拉扯。
3. 颈部入口/出口都要做倒角与防磨护套。

## 7. 底座壳体设计

### 7.1 底座结构分仓
1. 前仓：电源输入、开关、保险丝、降压模块。
2. 中仓：树莓派与 PCA9685。
3. 后仓：线束汇流与预留扩展位。

### 7.2 接口面板
1. 左侧：电源输入口、总开关。
2. 后侧：树莓派 USB/网口/HDMI 维护窗口。
3. 右侧：预留调试口（可封堵）。

### 7.3 散热与电磁兼容
1. 树莓派上方开散热孔，孔径 3mm，阵列排布。
2. 舵机电源模块与音频模块距离建议 > 20mm。
3. 功率线与信号线分侧布线，交叉处尽量垂直。

## 8. 线束与连接器方案

### 8.1 线束分组
1. 动力线组：6V 舵机电源。
2. 主控供电组：Pi 电源线。
3. 控制信号组：I2C、SPI、PWM。
4. 多媒体组：CSI 排线、USB 音频线。

### 8.2 连接器建议
1. 舵机：标准 3Pin 舵机头（建议增加锁扣延长线）。
2. 电源：XT30 或螺丝端子，按电流规格选择。
3. 头部快拆：JST-XH 系列多路插头。

### 8.3 固定策略
1. 每 60 到 80mm 设置一次线夹固定点。
2. 颈部入口必须加应力释放件。
3. 禁止线束悬空跨越舵机转动平面。

## 9. 材料与制造工艺

### 9.1 3D 打印建议
1. 材料：PETG（优先）或 ABS。
2. 层高：0.2mm。
3. 外壳填充：20% 到 25%。
4. 云台受力件填充：40% 到 60%。
5. 打印方向优先让受力方向垂直层纹，降低分层开裂风险。

### 9.2 CNC/钣金替代
1. 受力连接板可改铝合金 CNC。
2. 底座底板可改 2mm 铝板，提升刚度与配重。

## 10. 装配工艺流程（实体部分）

### 10.1 预装阶段
1. 先完成所有壳体去毛刺与铜螺母热压。
2. 检查所有孔位与模块干涉。
3. 空载试装双舵机与支架，确认无卡滞。

### 10.2 总装阶段
1. 底座内先安装电源与树莓派，再安装驱动板。
2. 固定云台总成并连接舵机线到 PCA9685。
3. 装头部内部模块，再与云台俯仰轴连接。
4. 整理线束后封闭后盖与底盖。

### 10.3 调整阶段
1. 通电前测量电源输出与对地短路。
2. 舵机回中后安装舵盘，避免初次撞壳。
3. 调整头部重心，使俯仰中位静止时不前倾。

## 11. 维护与扩展设计

### 11.1 维护口
1. 底部预留检修盖，可快速访问电源与线束。
2. 后盖可拆，支持相机/OLED/麦克风更换。

### 11.2 扩展位
1. 底座预留 1 个 50 x 40 x 20mm 扩展仓（后续传感器板）。
2. 预留 2 路 GPIO 走线孔用于新外设。

## 12. 实体设计验收标准

### 12.1 结构验收
1. 装配完成后壳体缝隙均匀，无明显翘曲。
2. 手动全行程转动无干涉、无刮线。
3. 头部在任意姿态保持时无明显松垮回落。

### 12.2 电气与线束验收
1. 开关、保险丝、接口可直接操作。
2. 线束无裸露压迫点，无高温贴近区。
3. 舵机动作时主控不掉电、不重启。

### 12.3 外观验收
1. 正面视觉中心（双眼+相机）对称误差小于 1mm。
2. 前面板无明显遮挡摄像头视场。
3. 设备放置桌面 10 分钟连续运行无异常振动。
